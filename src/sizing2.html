Perbandingan max-width Tailwind

max-w-[300px]
Konten ini dibatasi maksimal 300px.
max-w-full
Mengisi lebar penuh parent-nya.
max-w-md
Ukuran default Tailwind (28rem / 448px).
max-w-1/2
Maksimal 50% dari parent.
max-w-fit
Ukurannya pas dengan isi konten saja.
max-w-none
Tidak dibatasi sama sekali.
max-w-screen
Sama dengan lebar layar (viewport).
max-w-[var(--custom-max)]
Gunakan nilai CSS variabel: 500px

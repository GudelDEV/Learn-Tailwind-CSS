First Letter & First Line

::first-letter & ::first-line

first-letter:* → style huruf pertama dalam sebuah block text.

first-line:* → style satu baris pertama dari sebuah block text.

first-line

Digunakan untuk memberi style hanya pada baris pertama.

Tailwind CSS sangat fleksibel dan mendukung pseudo-class serta pseudo-element secara langsung, sehingga developer bisa membuat desain modern lebih efisien.

➡️ Baris pertama akan uppercase, punya spacing lebih lebar, dan berwarna abu-abu, sedangkan baris berikutnya normal.

first-letter

Digunakan untuk memberi style hanya pada huruf pertama.

Tailwind CSS adalah utility-first framework yang sangat powerful untuk membangun UI modern dengan cepat dan konsisten.

➡️ Huruf T akan besar, bold, dan biru, seperti gaya drop-cap di artikel majalah.

Kommbinasi First Letter dan First Line

Belajar Tailwind CSS sangat menyenangkan karena menyediakan banyak sekali utility class termasuk dukungan pseudo-element seperti first-line dan first-letter.

Placeholder

placeholder:* → untuk memberi style pada placeholder input.

➡️ Placeholder akan berwarna abu-abu dan italic, jadi lebih mudah dibaca.

Selection

::selection → untuk memberi style pada teks yang dipilih (highlight).

So I started to walk into the water. I won't lie to you boys, I was terrified. But I pressed on, and as I made my way past the breakers a strange calm came over me. I don't know if it was divine intervention or the kinship of all living things but I tell you Jerry at that moment, I was a marine biologist.

➡️ Teks yang dipilih akan punya background fuchsia dan teks berwarna fuchsia gelap.

File Input

::file → untuk memberi style pada input file.

➡️ Input file dengan style khusus, jadi lebih menarik dan mudah digunakan.

Marker

::marker → untuk memberi style pada marker list (bullet/number).

  • Item 1
  • Item 2
  • Item 3

➡️ Marker list akan berwarna sesuai style yang diberikan, jadi lebih menarik.

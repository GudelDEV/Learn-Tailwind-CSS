Cara Customisasion Di tailwind

Tailwind CSS menyediakan cara untuk mengcustomisasi class yang ada di Tailwind CSS. Kita bisa mengcustomisasi class yang ada di Tailwind CSS dengan cara membuat file tailwind.config.js di root project kita.

Contoh 1

h-[value] : untuk mengcustomisasi Height

w-[value] : untuk mengcustomisasi Width

bg-[value] : untuk megustomisasi bacgkround

max-w-[value]: Custom max width

min-w-[value]: Custom min width

p-[value], m-[value]: Custom padding/margin

rounded-[value]: Custom border radius

text-[value] fontsize

text-[color] color

border-[color]

shadow-[value]

2. Atur Tema dengan @theme

Misalnya:
@import "tailwindcss";

@theme {
  --breakpoint-xs: 30rem;
  --breakpoint-sm: 40rem;
  --color-primary: #1D4ED8;
  --spacing: 0.25rem;
  /* dan lainnya */
}

Ini menggantikan tailwind.config.js dan membuat konfigurasi lebih intuitif dan ditempatkan di satu file CSS
